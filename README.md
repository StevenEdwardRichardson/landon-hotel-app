# Landon Hotel Reservation App

Course project completed as part of WGU's B.S. Computer Science program (Advanced Java — D387).

## Overview
A multithreaded Spring Boot + Angular application modified to support internationalization, multi-timezone scheduling, and containerized deployment.

## Features
- English/French localization using Java ResourceBundles on separate threads
- REST endpoint converting presentation times across ET, MT, and UTC using java.time
- Multi-stage Dockerfile packaging the full Spring + Angular stack into a single deployable image

## Tech Stack
Spring Boot · Angular · Docker · Java · TypeScript · REST APIs
